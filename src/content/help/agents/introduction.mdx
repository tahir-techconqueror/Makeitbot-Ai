## What Are AI Agents?

AI agents are specialized AI assistants that handle specific tasks for your business. Instead of having one generic AI, Markitbot gives you a team of experts - each with their own specialty, personality, and tools.

<Callout type="info">
**Think of it like hiring a team:** You wouldn't hire one person to do marketing, compliance, analytics, and customer service. Each role needs expertise. That's why Markitbot has specialized agents for each area.
</Callout>

## How Agents Work Together (The "Hive Mind")

Agents don't work in isolation - they collaborate and share information:

### Example Workflow: New Product Launch

1. **You**: Add new product to catalog
2. **Ember**: Starts recommending it to customers
3. **Drip**: Creates launch campaign with compliance-checked copy
4. **Sentinel**: Validates all marketing claims are legal
5. **Radar**: Checks competitor pricing for positioning
6. **Ledger**: Suggests optimal launch price
7. **Pulse**: Tracks campaign performance
8. **Mrs. Parker**: Adds to loyalty program offers

All automatic. All working together.

<Callout type="success">
**The agents share a memory system** called "Letta" - when one agent learns something useful, the others can access that knowledge. It's like having a team that actually communicates!
</Callout>

- View agent activity logs and metrics

### For Dispensary Staff

## Common Agent Tasks

### Ask an Agent a Question
1. Go to **Dashboard ‚Üí Agents**
2. Click the agent you need
3. Type your question or request
4. Agent responds with answer and/or takes action

### Create a Campaign (Drip)
1. Click **Drip - Marketing Automation**
2. Choose campaign type (SMS, Email, Social)
3. Describe what you want
4. Drip drafts content and gets Sentinel's approval
5. Review and send

### Track Competitors (Radar)
1. Click **Radar - Competitive Intelligence**
2. Add competitors to track
3. Radar monitors their menus and prices
4. Get alerts when threats detected

### Analyze Performance (Pulse)
1. Click **Pulse - Data Analyst**
2. Ask about specific metrics
3. Pulse pulls data and provides insights
4. Request forecasts or reports

## Agent Best Practices

<Callout type="tip">
**Be specific**: Instead of "help with marketing," say "create a 4/20 sale campaign for flower products"

**Let agents collaborate**: Don't micromanage - agents will pull in other specialists as needed

**Review before launching**: Agents draft content, but you approve final campaigns

**Check agent logs**: See what agents are doing in the background
</Callout>

## Troubleshooting

<Callout type="warning">
**Issue**: Agent not responding

**Solutions**:
- Check agent status (may be in Training mode)
- Verify your plan includes that agent
- Try refreshing the page
- Check your internet connection
</Callout>

<Callout type="warning">
**Issue**: Agent gave wrong answer

**Solution**:
- Provide more context in your question
- Check agent's data sources are up to date
- Report the issue so agent can learn
</Callout>
