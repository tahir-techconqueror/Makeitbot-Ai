## Loyalty Program Types

### 1. Points-Based Program

**How it works:**
- Customers earn points on purchases
- Points can be redeemed for discounts
- Simple, easy to understand

**Example:**
```
Earn: 1 point per $1 spent
Redeem: 100 points = $5 off

Customer spends $50 â†’ Earns 50 points
After 2 purchases ($100 total) â†’ 100 points = $5 off
```

**Best for:**
- Most dispensaries (universal appeal)
- Easy to explain
- Flexible rewards

### 2. Tiered Program

**How it works:**
- Customers unlock tiers based on spending
- Each tier offers increasing benefits
- Motivates customers to spend more

**Example:**
```
Bronze (0-$499 lifetime):
- 1 point per $1
- Birthday 10% off

Silver ($500-$999 lifetime):
- 1.5 points per $1
- Birthday 15% off
- Early access to new products

Gold ($1,000+ lifetime):
- 2 points per $1
- Birthday 20% off
- Exclusive products
- Free delivery
```

**Best for:**
- Brands with wide product range
- Encouraging high-value customers
- Creating exclusivity

### 3. Cashback Program

**How it works:**
- Customers earn cashback on purchases
- Cashback automatically applies to next order

**Example:**
```
Earn: 5% cashback on all purchases

Customer spends $100 â†’ Earns $5 cashback
Next purchase: Cashback applied automatically
```

**Best for:**
- Simplicity (customers understand cash)
- No point calculations
- Automatic rewards

### 4. Referral Program

**How it works:**
- Customers refer friends
- Both get rewards when friend makes purchase

**Example:**
```
Refer a friend:
- Friend gets 15% off first purchase
- You get 100 points ($10 value)

Unlimited referrals!
```

**Best for:**
- Acquiring new customers
- Word-of-mouth marketing
- Growing customer base

**Optimize based on data:**

- Add more redemption options
- Remind customers of balance

**If redemption rate too high (>50%):**
- Increase redemption threshold
- Cap redemption per order
- Reduce point earning rate

**If enrollment low:**
- Increase sign-up bonus (200 â†’ 300 points)
- Promote more heavily
- Simplify sign-up process
