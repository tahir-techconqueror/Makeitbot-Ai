## Anatomy of a Great Description

### Essential Elements

**Every description should include:**

1. **Strain/Product overview** (1-2 sentences)
2. **Effects** (what will customer feel?)
3. **Flavors & aromas** (taste profile)
4. **THC/CBD content** (potency)
5. **Best use cases** (when to use)
6. **Warnings** (if applicable)

### Example: Excellent Description

```
Blue Dream 3.5g

Blue Dream is a legendary Sativa-dominant hybrid (60/40)
known for its balanced, euphoric effects. Born in California,
this strain crosses Blueberry Indica with Sativa Haze,
creating a perfect blend of relaxation and gentle cerebral
invigoration.

Effects: Uplifting, creative, focused, relaxed
Perfect for: Daytime use, creative projects, socializing
Flavors: Sweet berry, citrus, herbal notes

With 24.5% THC and 0.5% CBD, Blue Dream delivers smooth,
balanced effects that both novice and experienced consumers
enjoy. Users report enhanced creativity, stress relief, and
gentle body relaxation without sedation.

Medical patients use Blue Dream for pain, depression, and
nausea. The balanced profile makes it versatile for various
conditions without overwhelming psychoactive effects.

Lineage: Blueberry Ã— Haze
Terpenes: Myrcene, Pinene, Caryophyllene
```

**Word count:** 135 words
**Reading level:** Easy to understand
**Compliance:** No health claims, responsible language

### Example: Poor Description

```

```

**Problems:**
- Too short (8 words)
- No useful information
- Doesn't help customer decide
- Ember can't recommend accurately

## Writing Tips

### Use Descriptive Language

**Effects - Good examples:**
- Uplifting and energizing
- Calming body relaxation
- Focused mental clarity
- Euphoric and creative
- Gentle sedation

**Effects - Avoid:**
- Gets you high
- Strong/weak
- Good/bad
- Amazing

**Flavors - Good examples:**
- Sweet berry and citrus notes
- Earthy with pine undertones
- Tropical fruit and mango
- Diesel fuel and lemon zest
- Vanilla cream with hints of spice

**Flavors - Avoid:**
- Tastes good
- Nice flavor
- Classic weed taste

### Be Specific

**Vague:**
```
Great for pain relief and stress.
```

**Specific:**
```
Medical patients report relief from chronic pain,
migraines, and stress-related tension. The balanced
cannabinoid profile provides therapeutic benefits
without overwhelming psychoactive effects.
```

**Vague:**
```
Perfect for daytime or nighttime.
```

**Specific:**
```
Ideal for afternoon use when you need focus without
sedation. The balanced effects allow productivity
while relieving stress and tension.
```

### Show, Don't Just Tell

**Tell:**
```
This strain is popular.
```

**Show:**
```
Award-winning strain with a 4.8-star rating from
over 500 customers. Consistently a top-5 seller.
```

**Tell:**
```
High quality product.
```

**Show:**
```
Lab-tested at 24.5% THC with a rich terpene profile
dominated by myrcene and pinene. Grown using organic
methods without pesticides.
```
