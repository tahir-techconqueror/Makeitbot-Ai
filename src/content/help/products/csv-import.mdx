### Best Practices

**Product names:**
- Include weight/size (Blue Dream 3.5g, not just Blue Dream)
- Be specific (Sour Diesel Cart 0.5g, not Sour Diesel)
- Consistent format (Blue Dream 3.5g, Sour Diesel 3.5g)

**Categories:**
Use exact category names:
- Flower
- Pre-Rolls
- Vapes
- Edibles
- Concentrates
- Tinctures
- Topicals
- CBD
- Accessories

**Prices:**
- Enter as numbers only (35.00, not $35.00)
- Use OTD (out-the-door) pricing with tax included
- Decimals with 2 places (35.00, not 35)

**THC/CBD:**
- Enter as percentage (24.5, not 24.5%)
- Use 0 if not applicable (edibles CBD: 0)
- One decimal place is sufficient

**Descriptions:**
- 50-200 characters recommended
- Focus on benefits and effects
- Include flavors/aromas
- Avoid health claims

## Troubleshooting

### Error: "Invalid CSV format"

**Cause:** File not saved as CSV

**Solution:**
1. Open file in Excel/Google Sheets
2. File â†’ Save As
3. Format: CSV (Comma delimited)
4. Save and re-upload

### Error: "Column headers don't match"

**Cause:** Column names different from template

**Solution:**
- Download fresh template
- Copy your data to template
- Or use field mapping feature

### Error: "Duplicate SKUs found"

**Cause:** Same SKU used for multiple products

**Solution:**
- Make SKUs unique (BD-001, BD-002, etc.)
- Or remove SKU column (auto-generated)

### Warning: "Missing descriptions"

**Not an error, but:**
- Products import successfully
- Can add descriptions later
- Or bulk-generate with AI

**To fix:**
```
Dashboard â†’ Products â†’ Bulk Actions
â˜‘ Select products with missing descriptions
[Generate Descriptions with AI]
```

### Import stuck or slow?

**Large imports (500+ products):**
- May take 5-10 minutes
- Don't close browser window
- Check email for completion notice

**If truly stuck:**
1. Cancel import
2. Split CSV into smaller files (100 products each)
3. Import in batches
